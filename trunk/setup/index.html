Vanilla 1 Setup

Before you begin

If you are creating a fresh installation of Vanilla, all you need to do is upload all of the Vanilla files to your server. Once you are finished uploading, open this file in your web browser and click the install link below.

Click here to install a completely fresh version of Vanilla

If you are upgrading from an existing version of Vanilla, here are a few things you should do first:

  • Back up your Database

    I cannot stress how important this step is. The upgrader will be performing structural changes on your database and you will not be able to just use the old Vanilla code with your database after you are finished upgrading. If something goes wrong with this upgrade procedure, you *must* be able to fall back to your old installation.

  • Back up your old Vanilla

    Download and save your old Vanilla files to your local machine.

  • Save your appg/settings.php file

    You will need a backup of this file available so we can import your old configuration settings into the new system. If you have already removed this file, do not panic. You will have the ability to reconfigure these settings during the upgrade.

When you are finished backing everything up, upload all of the Vanilla files to your server, open this file in your web browser and click the upgrade link below.

Click here to upgrade your old version of Vanilla
