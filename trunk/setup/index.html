Upgrading or Installing Fresh?

If you are attempting to install a new version of Vanilla, you've come to the right place...

If you are attempting to upgrade your existing version of Vanilla, there are a few things you will need to do before you continue:

  • BACK UP YOUR DATABASE

    I cannot stress how important this step is. The upgrader will be performing structural changes on your database and you will not be able to just use the old Vanilla code with your database after you are finished upgrading. If something goes wrong with this upgrade procedure, you *must* be able to fall back to your old installation.

  • Save your appg/settings.php file

    You will need a backup of this file available so we can import your old configuration settings into the new system.

    If you have already removed this file, do not panic. You will have the ability to reconfigure these settings during the upgrade.
